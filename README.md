# Rally

Juego de rally interactivo con preguntas, códigos QR y panel de administración.

## Requisitos

- .NET 9.0
- SQL Server o SQLite

## Instalación

1. Restaurar dependencias:
```bash
dotnet restore
```

2. Compilar:
```bash
dotnet build
```

3. Ejecutar:
```bash
dotnet run
```
